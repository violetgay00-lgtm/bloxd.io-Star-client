FAQ

Is Star Client bannable?

Use at your own risk. We recommend private lobbies or alt accounts.

Does it auto update?

Yes. Tampermonkey pulls the script from GitHub, so you always run the latest build.

How to change keybinds?

Open the GUI, head to “Settings → Keybinds” and capture new keys per module.
